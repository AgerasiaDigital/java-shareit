# ShareIt — Сервис аренды вещей

## Описание

ShareIt — приложение для аренды вещей между пользователями. Позволяет размещать вещи для аренды, бронировать их, оставлять отзывы и создавать запросы на нужные предметы.

## Архитектура

Проект состоит из двух микросервисов:

- **Gateway** (порт `8080`) — валидация входящих запросов и проксирование на сервер
- **Server** (порт `9090`) — бизнес-логика, работа с БД

Оба сервиса разворачиваются через Docker Compose вместе с PostgreSQL.

```
Клиент → Gateway (8080) → Server (9090) → PostgreSQL
```

## Запуск

```bash
# Сборка
mvn clean package -DskipTests

# Запуск через Docker Compose
docker compose up --build
```

После запуска сервисы доступны по адресам:

| Сервис | Адрес |
|--------|-------|
| Gateway | `http://localhost:8080` |
| Server | `http://localhost:9090` |
| PostgreSQL | `localhost:6541` |

## Функциональность

### Пользователи

- Создание, обновление, получение и удаление пользователей

### Вещи

- Добавление и редактирование вещей
- Просмотр вещей владельца с информацией о бронированиях
- Поиск по названию и описанию
- Добавление комментариев после завершённой аренды

### Бронирования

- Создание бронирования с проверкой пересечений по датам
- Подтверждение или отклонение владельцем вещи
- Фильтрация по статусу: `ALL`, `CURRENT`, `PAST`, `FUTURE`, `WAITING`, `REJECTED`

### Запросы на вещи

- Создание запроса на нужную вещь
- Просмотр своих запросов с откликами
- Просмотр запросов других пользователей с пагинацией

## API

### Пользователи `/users`

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/users` | Создать пользователя |
| `PATCH` | `/users/{id}` | Обновить пользователя |
| `GET` | `/users/{id}` | Получить пользователя |
| `DELETE` | `/users/{id}` | Удалить пользователя |
| `GET` | `/users` | Получить всех пользователей |

### Вещи `/items`

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/items` | Добавить вещь |
| `PATCH` | `/items/{id}` | Обновить вещь |
| `GET` | `/items/{id}` | Получить вещь |
| `GET` | `/items` | Вещи владельца |
| `GET` | `/items/search?text=` | Поиск вещей |
| `POST` | `/items/{id}/comment` | Добавить комментарий |

### Бронирования `/bookings`

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/bookings` | Создать бронирование |
| `PATCH` | `/bookings/{id}?approved=` | Подтвердить / отклонить |
| `GET` | `/bookings/{id}` | Получить бронирование |
| `GET` | `/bookings?state=` | Бронирования пользователя |
| `GET` | `/bookings/owner?state=` | Бронирования владельца |

### Запросы на вещи `/requests`

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/requests` | Создать запрос |
| `GET` | `/requests` | Свои запросы |
| `GET` | `/requests/all?from=&size=` | Запросы других пользователей |
| `GET` | `/requests/{id}` | Получить запрос по id |

Все запросы, требующие идентификации пользователя, передают заголовок `X-Sharer-User-Id`.
